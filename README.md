```
ملفات المتجر + ملفات المنافسين
         ↓
   محرك مهووس v9.0
   (FAISS + RapidFuzz + Gemini AI)
         ↓
  ملف CSV جاهز للرفع على سلة
```

---

## 🚀 طريقة الاستخدام اليومي (3 خطوات فقط)

### الخطوة 1: رفع ملفات البيانات

| المجلد | ما تضعه فيه |
|:---|:---|
| `input/store/` | ملف متجر مهووس (CSV) |
| `input/competitors/` | ملفات المنافسين (CSV) — يمكن أكثر من ملف |
| `input/brands/` | ملف الماركات `ماركاتمهووس.csv` (اختياري) |

**كيف ترفع الملفات؟**
1. اذهب إلى المستودع على GitHub
2. افتح المجلد المطلوب (مثلاً `input/competitors/`)
3. اضغط **"Add file"** → **"Upload files"**
4. اسحب الملف وأفلته → اضغط **"Commit changes"**

---

### الخطوة 2: تشغيل المحرك

**التشغيل اليدوي (عند الحاجة):**
1. اذهب إلى تبويب **"Actions"** في المستودع
2. اضغط على **"محرك مهووس للحسم v9.0"**
3. اضغط **"Run workflow"** → اختر تفعيل الذكاء الاصطناعي أم لا
4. اضغط **"Run workflow"** الأخضر

**التشغيل التلقائي:**
- يعمل تلقائياً **كل يوم الساعة 3 صباحاً** (توقيت السعودية)
- يعمل تلقائياً **عند رفع ملفات جديدة** في مجلد `input/`

---

### الخطوة 3: تحميل النتائج

1. بعد اكتمال التشغيل (3-5 دقائق)، اذهب إلى **"Actions"**
2. اضغط على آخر تشغيل ناجح ✅
3. في أسفل الصفحة، ابحث عن **"Artifacts"**
4. اضغط على **"mahwous-results-..."** لتحميل ملف ZIP يحتوي على:

| الملف | المحتوى |
|:---|:---|
| `سلة_فرص_جديدة_YYYY-MM-DD.csv` | **جاهز للرفع المباشر على سلة** |
| `فرص_جديدة_YYYY-MM-DD.csv` | تفاصيل الفرص مع نسبة الثقة |
| `مكررات_محظورة_YYYY-MM-DD.csv` | المنتجات المحظورة (مرجع) |
| `مراجعة_يدوية_YYYY-MM-DD.csv` | حالات تحتاج قرار بشري |

---

## 🔑 إعداد مفاتيح الذكاء الاصطناعي (مرة واحدة فقط)

1. اذهب إلى **Settings** → **Secrets and variables** → **Actions**
2. اضغط **"New repository secret"**
3. أضف المفاتيح التالية:

| الاسم | القيمة | الغرض |
|:---|:---|:---|
| `GEMINI_API_KEY` | مفتاحك من Google AI Studio | تصفية المنطقة الرمادية |
| `OPENAI_API_KEY` | مفتاحك من OpenAI (اختياري) | بديل للذكاء الاصطناعي |

> **ملاحظة:** Gemini 1.5 Flash مجاني (مليون Token يومياً) — احصل على مفتاحك من [Google AI Studio](https://aistudio.google.com/)

## 🧠 المعادلة الذهبية للمطابقة

```
نسبة التطابق = (الاسم الصافي × 40%) + (الماركة × 30%) + (المواصفات × 20%) + (البصمة البصرية × 10%)
```

| النسبة | القرار |
|:---:|:---|
| ≥ 88% | مكرر محظور |
| 55% - 87% | منطقة رمادية (يراجعها الذكاء الاصطناعي) |
| < 55% | فرصة جديدة (بعد التحقق العكسي الثلاثي) |
